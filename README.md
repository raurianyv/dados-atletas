 Projeto Dados de Atletas
Aplicação desenvolvida em JavaScript para receber informações de um atleta, calcular parâmetros físicos e estatísticos e exibir os resultados no console.

 Objetivo
Criar uma classe Atleta capaz de:

Armazenar dados do atleta
Calcular a categoria por idade
Calcular o IMC
Calcular a média válida das notas
Exibir todas as informações no console

Categoria por idade
Faixa Etária	Categoria
9 a 11 anos	Infantil
12 e 13 anos	Juvenil
14 e 15 anos	Intermediário
16 a 30 anos	Adulto
Demais idades	Sem categoria

Cálculo do IMC
Fórmula utilizada:
IMC = peso / (altura x altura)
